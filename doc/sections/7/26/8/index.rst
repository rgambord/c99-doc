7.26.8 Integer types \<stdint.h>
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. _9899_7.26.8p1:

.. container:: snum

   :ref:`1 <9899_7.26.8p1>`

Typedef names beginning with int or uint and ending with \_t may be added to the types defined in the :ref:`\<stdint.h> <9899_7.18>` header. Macro names beginning with INT or UINT and ending with \_MAX, \_MIN, or \_C may be added to the macros defined in the :ref:`\<stdint.h> <9899_7.18>` header.
