J.3.5 Integers
^^^^^^^^^^^^^^

.. _9899_J.3.5p1:

:ref:`1 <9899_J.3.5p1>`

-  Any extended integer types that exist in the implementation (:ref:`6.2.5 <9899_6.2.5>`).
-  Whether signed integer types are represented using sign and magnitude, two's complement, or ones' complement, and whether the extraordinary value is a trap representation or an ordinary value (:ref:`6.2.6.2 <9899_6.2.6.2>`).
-  The rank of any extended integer type relative to another extended integer type with the same precision (:ref:`6.3.1.1 <9899_6.3.1.1>`).
-  The result of, or the signal raised by, converting an integer to a signed integer type when the value cannot be represented in an object of that type (:ref:`6.3.1.3 <9899_6.3.1.3>`).
-  The results of some bitwise operations on signed integers (:ref:`6.5 <9899_6.5>`).
