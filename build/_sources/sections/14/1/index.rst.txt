.. _9899_G.1:

G.1 Introduction
~~~~~~~~~~~~~~~~

.. _9899_G.1p1:

.. container:: snum

   :ref:`1 <9899_G.1p1>`

This annex supplements :ref:`annex F <9899_F>` to specify complex arithmetic for compatibility with IEC 60559 real floating-point arithmetic. Although these specifications have been carefully designed, there is little existing practice to validate the design decisions. Therefore, these specifications are not normative, but should be viewed more as recommended practice. An implementation that defines \__STDC_IEC_559_COMPLEX\_\_ should conform to the specifications in this annex.
