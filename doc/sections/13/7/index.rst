.. _9899_F.7p1:

:ref:`1 <9899_F.7p1>` The floating-point environment defined in :ref:`\<fenv.h> <9899_7.6>` includes the IEC 60559 floating- point exception status flags and directed-rounding control modes. It includes also IEC 60559 dynamic rounding precision and trap enablement modes, if the implementation supports them.\ [#9899_note312]_





.. rubric:: Footnotes

.. [#9899_note312] This specification does not require dynamic rounding precision nor trap enablement modes.
