7.12.4.7 The tan functions

Synopsis

1

#include <math.h>
double tan(double x);
float tanf(float x);
long double tanl(long double x);

Description

2

The tan functions return the tangent of x (measured in radians).

Returns

3

The tan functions return tan x.
