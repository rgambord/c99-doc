7.21.4.2 The strcmp function
''''''''''''''''''''''''''''

.. rubric:: Synopsis

.. _9899_7.21.4.2p1:

:ref:`1 <9899_7.21.4.2p1>`

::

    #include <string.h>
    int strcmp(const char *s1, const char *s2);

.. rubric:: Description

.. _9899_7.21.4.2p2:

:ref:`2 <9899_7.21.4.2p2>` The strcmp function compares the string pointed to by s1 to the string pointed to by s2.

.. rubric:: Returns

.. _9899_7.21.4.2p3:

:ref:`3 <9899_7.21.4.2p3>` The strcmp function returns an integer greater than, equal to, or less than zero, accordingly as the string pointed to by s1 is greater than, equal to, or less than the string pointed to by s2.
