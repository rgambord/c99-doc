6.3 Conversions
~~~~~~~~~~~~~~~


.. toctree::
   :maxdepth: 2

   1/index
   2/index


.. _9899_6.3p1:

:ref:`1 <9899_6.3p1>` Several operators convert operand values from one type to another automatically. This subclause specifies the result required from such an implicit conversion, as well as those that result from a cast operation (an explicit conversion). The list in :ref:`6.3.1.8 <9899_6.3.1.8>` summarizes the conversions performed by most ordinary operators; it is supplemented as required by the discussion of each operator in :ref:`6.5 <9899_6.5>`.

.. _9899_6.3p2:

:ref:`2 <9899_6.3p2>` Conversion of an operand value to a compatible type causes no change to the value or the representation.
