.. _9899_7.19.10.1p2:

.. container:: snum

   :ref:`2 <9899_7.19.10.1p2>`

The clearerr function clears the end-of-file and error indicators for the stream pointed to by stream.

.. rubric:: Returns

.. _9899_7.19.10.1p3:

.. container:: snum

   :ref:`3 <9899_7.19.10.1p3>`

The clearerr function returns no value.
