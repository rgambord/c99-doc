7.3.6.5 The csinh functions
'''''''''''''''''''''''''''

.. rubric:: Synopsis

.. _9899_7.3.6.5p1:

:ref:`1 <9899_7.3.6.5p1>`

::

    #include <complex.h>
    double complex csinh(double complex z);

.. rubric:: Description

.. _9899_7.3.6.5p2:

:ref:`2 <9899_7.3.6.5p2>` The csinh functions compute the complex hyperbolic sine of z.

.. rubric:: Returns

.. _9899_7.3.6.5p3:

:ref:`3 <9899_7.3.6.5p3>` The csinh functions return the complex hyperbolic sine value.
