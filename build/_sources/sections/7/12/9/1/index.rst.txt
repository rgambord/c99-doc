.. rubric:: Description

.. _9899_7.12.9.1p2:

.. container:: snum

   :ref:`2 <9899_7.12.9.1p2>`

The ceil functions compute the smallest integer value not less than x.

.. rubric:: Returns

.. _9899_7.12.9.1p3:

.. container:: snum

   :ref:`3 <9899_7.12.9.1p3>`

The ceil functions return ⌈x⌉, expressed as a floating-point number.
