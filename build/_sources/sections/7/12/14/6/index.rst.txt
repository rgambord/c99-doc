.. rubric:: Description

.. _9899_7.12.14.6p2:

.. container:: snum

   :ref:`2 <9899_7.12.14.6p2>`

The isunordered macro determines whether its arguments are unordered.

.. rubric:: Returns

.. _9899_7.12.14.6p3:

.. container:: snum

   :ref:`3 <9899_7.12.14.6p3>`

The isunordered macro returns 1 if its arguments are unordered and 0 otherwise.
