3.1 access
~~~~~~~~~~

.. _9899_3.1p1:

:ref:`1 <9899_3.1p1>` access
    〈execution-time action〉 to read or modify the value of an object

.. _9899_3.1p2:

    :ref:`2 <9899_3.1p2>` NOTE 1 Where only one of these two actions is meant, ''read'' or ''modify'' is used.

.. _9899_3.1p3:

    :ref:`3 <9899_3.1p3>` NOTE 2 "Modify'' includes the case where the new value being stored is the same as the previous value.

.. _9899_3.1p4:

    :ref:`4 <9899_3.1p4>` NOTE 3 Expressions that are not evaluated do not access objects.
