3.4.1 implementation-defined behavior
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. _9899_3.4.1p1:

:ref:`1 <9899_3.4.1p1>` implementation-defined behavior
    unspecified behavior where each implementation documents how the choice is made

.. _9899_3.4.1p2:

    :ref:`2 <9899_3.4.1p2>` EXAMPLE An example of implementation-defined behavior is the propagation of the high-order bit when a signed integer is shifted right.
