H.2.2.1 Integer operations
''''''''''''''''''''''''''

.. _9899_H.2.2.1p1:

:ref:`1 <9899_H.2.2.1p1>` The integer operations on integer types are the following:

.. code-block:: text

    addI           x + y
    subI           x - y
    mulI           x * y
    divI, divtI    x / y
    remI, remtI    x % y
    negI           -x
    absI           abs(x), labs(x), llabs(x)
    eqI            x == y
    neqI           x != y
    lssI           x < y
    leqI           x <= y
    gtrI           x > y
    geqI           x >= y

where x and y are expressions of the same integer type.
