J.5.1 Environment arguments
^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. _9899_J.5.1p1:

:ref:`1 <9899_J.5.1p1>` In a hosted environment, the main function receives a third argument, char \*envp[], that points to a null-terminated array of pointers to char, each of which points to a string that provides information about the environment for this execution of the program (:ref:`5.1.2.2.1 <9899_5.1.2.2.1>`).
