G.4.2 Real and imaginaryÔÉÅ

1

When a value of imaginary type is converted to a real type other than _Bool,[1] the result is a positive zero.

2

When a value of real type is converted to an imaginary type, the result is a positive imaginary zero.

Footnotes

[1]

See 6.3.1.2.
