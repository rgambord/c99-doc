7.12.14.6 The isunordered macro
'''''''''''''''''''''''''''''''

.. rubric:: Synopsis

.. _9899_7.12.14.6p1:

:ref:`1 <9899_7.12.14.6p1>`

::

    #include <math.h>
    int isunordered(real-floating x, real-floating y);

.. rubric:: Description

.. _9899_7.12.14.6p2:

:ref:`2 <9899_7.12.14.6p2>` The isunordered macro determines whether its arguments are unordered.

.. rubric:: Returns

.. _9899_7.12.14.6p3:

:ref:`3 <9899_7.12.14.6p3>` The isunordered macro returns 1 if its arguments are unordered and 0 otherwise.
