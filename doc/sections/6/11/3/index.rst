6.11.3 External names
^^^^^^^^^^^^^^^^^^^^^

.. _9899_6.11.3p1:

:ref:`1 <9899_6.11.3p1>` Restriction of the significance of an external name to fewer than 255 characters (considering each universal character name or extended source character as a single character) is an obsolescent feature that is a concession to existing implementations.
