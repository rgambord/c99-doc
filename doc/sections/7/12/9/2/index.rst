.. _9899_7.12.9.2p2:

.. container:: snum

   :ref:`2 <9899_7.12.9.2p2>`

The floor functions compute the largest integer value not greater than x.

.. rubric:: Returns

.. _9899_7.12.9.2p3:

.. container:: snum

   :ref:`3 <9899_7.12.9.2p3>`

The floor functions return ⌊x⌋, expressed as a floating-point number.
