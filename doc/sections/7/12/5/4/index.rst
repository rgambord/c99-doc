7.12.5.4 The cosh functions
'''''''''''''''''''''''''''

.. rubric:: Synopsis

.. _9899_7.12.5.4p1:

:ref:`1 <9899_7.12.5.4p1>`

::

    #include <math.h>
    double cosh(double x);
    float coshf(float x);
    long double coshl(long double x);

.. rubric:: Description

.. _9899_7.12.5.4p2:

:ref:`2 <9899_7.12.5.4p2>` The cosh functions compute the hyperbolic cosine of x. A range error occurs if the magnitude of x is too large.

.. rubric:: Returns

.. _9899_7.12.5.4p3:

:ref:`3 <9899_7.12.5.4p3>` The cosh functions return cosh x.
