.. _9899_5p1:

:ref:`1 <9899_5p1>` An implementation translates C source files and executes C programs in two data- processing-system environments, which will be called the translation environment and the execution environment in this International Standard. Their characteristics define and constrain the results of executing conforming C programs constructed according to the syntactic and semantic rules for conforming implementations.
