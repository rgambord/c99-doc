F.9.3.7 The log functions
'''''''''''''''''''''''''

.. _9899_F.9.3.7p1:

:ref:`1 <9899_F.9.3.7p1>`

-  log((+-)0) returns -(inf) and raises the ''divide-by-zero'' floating-point exception.
-  log(1) returns +0.
-  log(x) returns a NaN and raises the ''invalid'' floating-point exception for x < 0.
-  log(+(inf)) returns +(inf).
