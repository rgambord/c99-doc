6.3.1.7 Real and complex
''''''''''''''''''''''''

.. _9899_6.3.1.7p1:

:ref:`1 <9899_6.3.1.7p1>` When a value of real type is converted to a complex type, the real part of the complex result value is determined by the rules of conversion to the corresponding real type and the imaginary part of the complex result value is a positive zero or an unsigned zero.

.. _9899_6.3.1.7p2:

:ref:`2 <9899_6.3.1.7p2>` When a value of complex type is converted to a real type, the imaginary part of the complex value is discarded and the value of the real part is converted according to the conversion rules for the corresponding real type.
