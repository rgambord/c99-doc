H.1 Introduction
~~~~~~~~~~~~~~~~

.. _9899_H.1p1:

:ref:`1 <9899_H.1p1>` This annex documents the extent to which the C language supports the ISO/IEC 10967-1 standard for language-independent arithmetic (LIA-1). LIA-1 is more general than IEC 60559 (:ref:`annex F <9899_F>`) in that it covers integer and diverse floating-point arithmetics.
