F.9.2.3 The atanh functions
'''''''''''''''''''''''''''

.. _9899_F.9.2.3p1:

:ref:`1 <9899_F.9.2.3p1>`

-  atanh((+-)0) returns (+-)0.
-  atanh((+-)1) returns (+-)(inf) and raises the ''divide-by-zero'' floating-point exception.
-  atanh(x) returns a NaN and raises the ''invalid'' floating-point exception for \| x \| > 1.
