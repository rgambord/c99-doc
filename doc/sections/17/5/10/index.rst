J.5.10 The asm keyword
^^^^^^^^^^^^^^^^^^^^^^

.. _9899_J.5.10p1:

.. container:: snum

   :ref:`1 <9899_J.5.10p1>`

The asm keyword may be used to insert assembly language directly into the translator output (:ref:`6.8 <9899_6.8>`). The most common implementation is via a statement of the form:

.. code-block:: text

    asm ( character-string-literal );
