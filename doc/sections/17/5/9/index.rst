J.5.9 The fortran keyword
^^^^^^^^^^^^^^^^^^^^^^^^^

.. _9899_J.5.9p1:

:ref:`1 <9899_J.5.9p1>` The fortran function specifier may be used in a function declaration to indicate that calls suitable for FORTRAN should be generated, or that a different representation for the external name is to be generated (:ref:`6.7.4 <9899_6.7.4>`).
