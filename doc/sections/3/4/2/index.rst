3.4.2 locale-specific behavior
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. _9899_3.4.2p1:

:ref:`1 <9899_3.4.2p1>` locale-specific behavior
    behavior that depends on local conventions of nationality, culture, and language that each implementation documents

.. _9899_3.4.2p2:

    :ref:`2 <9899_3.4.2p2>` EXAMPLE An example of locale-specific behavior is whether the islower function returns true for characters other than the 26 lowercase Latin letters.
