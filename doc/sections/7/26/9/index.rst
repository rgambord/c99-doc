7.26.9 Input/output \<stdio.h>
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. _9899_7.26.9p1:

:ref:`1 <9899_7.26.9p1>` Lowercase letters may be added to the conversion specifiers and length modifiers in fprintf and fscanf. Other characters may be used in extensions.

.. _9899_7.26.9p2:

:ref:`2 <9899_7.26.9p2>` The gets function is obsolescent, and is deprecated.

.. _9899_7.26.9p3:

:ref:`3 <9899_7.26.9p3>` The use of ungetc on a binary stream where the file position indicator is zero prior to the call is an obsolescent feature.
